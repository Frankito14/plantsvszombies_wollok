## Reglas de Juego / Instrucciones
Defender tu casa de los zombies usando las plantas para frenar su avance

## Controles
- Comenzar partida: Enter
- Plantar: Espacio
- Mover seleccionador de plantas: [Q][E]
- Mover Cabezal: Flechas direccionales
